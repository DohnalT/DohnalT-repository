Úvod

Semestrální práce z předmětu "Značkovací jazyky" se zabývá tvorbou XML dokumentu popisujícího soubor do kterého smyšlená aplikace ukládá svá nastavení. Součástí je také tvorba XSD schéma, XSL transformace do JSNO a tato dokumentace. Dokument byl tvořen jako konfigurační soubor hudebního přehrávače.

Metodika

Všechny dokumenty byly vytvořeny v programu netbeans.

Nejprve byl vytvořen hrubý XML dokument se jmény elementů a atributů. Ten byl následně ručně konverttován do XSD podoby.

Validita těchto dvou dokumentů byla zkontrolována pomocí webového validátoru . Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu.

XSL transformace do JSON byla psána ručně. Data jsou strukturována do JSON tabulek podle kategorií vyplývajících z designu XSD dokumentu. Do XML dokumentu byl doplněn odkaz na tuto transformaci. Při testování bylo zjištěno že prohlížeče Google Chrome ani Opera neumí s transformacemi pracovat (zobrazí prázdnou stránku). Transformace od JSON neobsahuje všechna data z XML souboru.

Celý projekt byl v průběhu jeho tvorby postupně nahráván (a verzován) na GitHubu.

Popis transformace

HTML

Data jsou transformována do HTML dokumentu. Jsou strukturována tak, jak by typicky bylo strukturováno menu Nastavení. Všechny atributy "enabled" jsou zabaleny v podmínkách, přes které se přepisují z true/false na On/Off. Transformace od HTML neobsahuje všechna data z XML souboru.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy. Jeho výhodou je dobrá čitelnost a šikoké možnosti filtrování a omezování dat a datovýh typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze textový editor.
